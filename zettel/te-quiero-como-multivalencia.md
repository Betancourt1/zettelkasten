# "Te quiero" como declaración multivalente

Rupi Kaur lo abre: "te quiero" puede significar el deseo de querer, quedarse un poco más, no saber cómo irse, o no tener otro lugar a donde ir. Cuatro frases, cuatro razones distintas para la misma palabra.

## Idea
- El amor no es un estado binario; puede ser intención, inercia, miedo, o falta de alternativas.
- Nombrar estas diferencias permite distinguir lo que realmente sucede en un vínculo.
