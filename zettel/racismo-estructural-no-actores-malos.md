# El racismo como estructura, no como actores malos

Los primeros sistemas de big data se construyeron eliminando humanos "sesgados" del proceso. Pero si el racismo es estructural, no individual, sacar las "manzanas podridas" no lo elimina: solo lo automatiza.

## Idea
- El "nuevo racismo" (Robin DiAngelo) cree que el problema son individuos malos, no sistemas.
- Los algoritmos pueden replicar discriminación estructural incluso sin intención humana explícita.
- La tecnología de datos hereda los sesgos de los datos que procesa, que reflejan desigualdades históricas.
