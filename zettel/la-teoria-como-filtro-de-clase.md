# La teoría como filtro de clase

Presentar a la clase trabajadora como incapaz de pensamiento teórico no describe una limitación intelectual: funciona como jerarquía moral.

## Idea
- Se naturaliza la diferencia de clase como si fuera una propiedad interna de las personas.
- El supuesto “desinterés teórico” opera como mecanismo de superioridad y exclusión.
